# Riffle

A Go application.

## Building

To build the application:

```bash
make
```

## Running

To run the application:

```bash
./bin/riffle
```

## Development

### Prerequisites

- Go 1.24 or later
- Make

### Building for different platforms

To build for a specific platform:

```bash
make PLATFORMS=linux/amd64
```

Available platforms:
- linux/amd64
- linux/arm64
- darwin/amd64
- darwin/arm64
